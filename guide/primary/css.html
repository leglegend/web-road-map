# CSS ⭐️

层叠样式表（Cascading Style Sheet，简称：CSS）是一种样式表语言，用来描述HTML文档的呈现。

# 一、引入方式

  1. 行内样式
<div style="height:100px"></div>
  1. 内部样式表
<head>
  <style>
    div {
      height:100px;
    }
  </style>
</head>
  1. 外部样式表
/* style.css */
body {
  margin: 0px;
}

# 二、选择器

# 标签选择器

/* 标签选择器 */
div { }
/* 标签属性选择器 */
a[title] { } 
/* 特定值的标签属性 */
a[href="https://example.com"] { }

# 类选择器

.class-name { }
/* 与下面的属性选择器等价 */
[class=class-name] { }

# id选择器

#canvas { }

# 通配符选择器

通配符*可以匹配任意HTML元素。

/* 为所有元素添加样式 */
* {
    font-size: 14px;
}
/* 为所有带有title属性的元素添加样式 */
a[title] { }

注意

不推荐使用通配选择器，因为它是性能最低的一个 CSS 选择器。
