# 前端入门

对于初学者，最重要的是先打好基础，前端最基础的知识就是HTML、CSS和JS，本节会罗列初学阶段应该掌握的知识，还有推荐一些学习资料。建议初学者看视频学习，边看边手动操作。

# 学习目标

能独自完成一个网页，HTML、CSS、JS配合使用搭建一个个人网站，能实现基本的展示、发送请求、处理数据、渲染页面等功能。
